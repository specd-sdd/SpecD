# @specd/mcp

MCP (Model Context Protocol) server adapter for [specd](../../README.md).

> **Status:** Stub -- not yet implemented.

This package will expose specd's core functionality as an MCP server, enabling AI tools and IDE extensions to interact with specd through the Model Context Protocol.

## Planned capabilities

- Spec browsing and context retrieval
- Change lifecycle management
- Schema and workspace operations
- Code graph queries

## Dependencies

- `@specd/core`
