# Exploration Context: complete-agent-plugins-codex-copilot-open-code

Generated: 2026-04-20

## Problem Statement

El usuario quiere terminar los plugins de agentes para `codex` y `copilot` y crear un plugin nuevo para `open code`, tomando como base el plugin existente de `claude`.

Objetivo funcional expresado por el usuario:

- Copiar el comportamiento del plugin de Claude.
- Cambiar nombres/descripciones/referencias por cada agente.
- Configurar rutas de instalación específicas por agente.
- Revisar/definir qué frontmatter soporta cada agente.

## Approach / Solution Outline

En la exploración se acordó un enfoque de paridad:

- Usar `packages/plugin-agent-claude` como implementación de referencia.
- Llevar `packages/plugin-agent-copilot` y `packages/plugin-agent-codex` desde stub a implementación real de instalación/desinstalación de skills con inyección de frontmatter.
- Añadir un nuevo plugin para Open Code (nuevo workspace y paquete), con la misma estructura base y frontmatter adaptado.
- Mantener las skills base sin frontmatter en `@specd/skills` y delegar inyección a cada plugin de agente (alineado con `skills:skill-templates-source`).
